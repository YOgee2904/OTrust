about

OTrust is a minimal, privacy-first authentication system.

It secures user access without creating global identities that follow people across applications. Each app receives a scoped user identity, making cross-application correlation impossible by design.

OTrust validates sessions, enforces trust boundaries, and gives users direct control over access — nothing more.
